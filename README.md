### Konfigurasi 

Anda diharuskan untuk melakukan konfigurasi pada env ini:
```.env
APP_DEBOUNCE_DELAY=0

SCOUT_DRIVER=database

DB_CONNECTION=mysql
DB_HOST=127.0.0.1
DB_PORT=3306
DB_DATABASE=laravel
DB_USERNAME=root
DB_PASSWORD=

PUSHER_APP_ID=
PUSHER_APP_KEY=
PUSHER_APP_SECRET=
PUSHER_APP_CLUSTER=mt1
```
